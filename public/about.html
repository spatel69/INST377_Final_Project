About This App

This app was developed to give users personalized clothing suggestions based on the weather in a specific city that they search. We use real-time data from Weatherstack, Open-meteo, and the National Weather Service so users can visualize current weather, air quality, future weather trends, and clothing recommendations with visual examples. Our mission is to help remove uncertainty associated with weather related attire and to make everyone's daily life more convenient. If you are traveling, don't know what clothes to wear, or simply just want to know what the weather is going to be in your area, then this app is for you!

Weatherstack API
We use this API to pull several data elements regarding the current weather in a specific city (Condition, Temperature, UV Index, Humidity, etc.). Given this data, we will give you clothing recommendations for the day
National Weather Service API
This API is used to display a forecast for the weather in a city over the next several hours
Open-meteo Air Quality API
This API is used to get the air quality for a city so that we can give you recommendations on whether to go outside or stay indoors
