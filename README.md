## Developer Manual

### Installation
1. Clone the Repository:

    ```
    git clone <Repo Link>

    cd INST377_Final_Project
    ```

2. Install dependencies:

    ```
    npm install
    ```

### Run the App
1. Begin by starting the server:
    ```
    npm start
    ```

2. App/Server will be hosted on:
    ```
    http://localhost:3005
    ```
